# Contributing to AetherisNET

Thanks for your interest in contributing! We welcome contributions of all sizes â€” from documentation fixes to new features and bug reports.

How to contribute
- Fork the repository and create a feature branch (e.g. `feature/cool-stuff`).
- Make your changes and run the linter: `pip install -r requirements.txt && flake8`.
- Open a Pull Request describing the change, why it's needed, and any testing you've done.

Code style
- Keep changes focused and small.
- Use clear commit messages and reference related issues in your PR.

Reporting bugs
- Open an issue with reproduction steps, expected vs actual behavior, and any relevant logs or screenshots.

Security
- If you discover a security vulnerability, please open a private issue or contact the maintainer directly instead of posting public details.
